Socket_Todolist
===============

**这是一个Todolist。\
利用Socket+SQLAlchemy+Ajax实现\
登录后：你可以新建和处理待办事项。**

---
基本功能：
+ 注册
+ 登陆
+ 新建任务
+ 标记完成
+ 删除
+ 退出
